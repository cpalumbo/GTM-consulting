# New Client Engagement Playbook

Step-by-step guide for running a full GTM consulting engagement.

## Pre-Engagement

### Qualification
Before accepting an engagement, confirm:
- Client has a product (or clear product vision)
- Client has budget and timeline commitment (minimum 3 months)
- Client has decision-making authority present in workshops
- The problem is a GTM problem (not a product problem)

### Scoping
Standard engagement options:
1. **Full GTM Strategy** (12-16 weeks): All 6 decisions + execution plan
2. **GTM Sprint** (4-6 weeks): Focused on 2-3 decisions (e.g., Market + ICP + Positioning)
3. **Single Workstream** (2-4 weeks): One deep dive (e.g., Pricing Strategy only)

## Engagement Phases

### Week 1-2: Discovery
- Run discovery workshop using GTM Strategy Agent prompts
- Audit existing materials (pitch deck, website, CRM data)
- Interview 3-5 customers if possible
- Document current state and gaps

### Week 2-4: Market & Customer
- Use `market-analyzer` for segment scoring and sizing
- Use `icp-profiler` for customer profiling
- Use `competitor-mapper` for competitive analysis
- Deliverables: Market Report, ICP Document, Competitive Landscape

### Week 4-6: Strategy
- Use `positioning-builder` for positioning workshop
- Use `pricing-strategist` for pricing analysis
- Product value proposition workshop
- Deliverables: Positioning Statement, Pricing Strategy, Value Prop Canvas

### Week 6-8: Execution Planning
- Use `growth-channel-selector` for channel strategy
- Use `pipeline-architect` for pipeline design
- Use `sales-qualification` for MEDDICC implementation
- Deliverables: Channel Plan, Pipeline Model, Qualification Framework

### Week 8-10: Implementation
- Use `gtm-engineering-stack` for tech stack recommendation
- Use `launch-planner` for first launch planning
- Synthesize GTM Canvas
- Deliverables: Tech Stack Plan, Launch Plan, GTM Canvas

### Week 10-12: Handoff & Review
- Present final GTM Canvas to stakeholders
- Train sales team on MEDDICC and battle cards
- Set up pipeline review cadence
- Define 90-day experiment plan
- Hand off all templates and frameworks

## Recurring Check-ins (Post-Engagement)
- Monthly: Pipeline review, channel performance, experiment results
- Quarterly: GTM strategy review, positioning refresh, competitive update
- Annually: Full GTM reassessment
